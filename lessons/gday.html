Let's say Gday!

As per usual, our first step is to say hello!

Why do we say G’day? It’s usually more difficult than the traditional hello! If you give the Python node a go, you may notice why. Also, we’re paying homage to our ⅓ Australian roots.


Code <!DOCTYPE html>
<html>
G’day Mate
</html>
gday.html
G'day Mate


There you have it! That’s your first ever webpage. Doesn’t it feel great? Perhaps not. We’ll work on it.
First, let’s understand what each line does.

Line Explanation
<!DOCTYPE html> This is the first thing that every HTML document should start with. It tells your web browser what version of html you are writing in. If this line is present it tells the browser to use HTML5.
<html> This is your first tag. It acts as a container for all the other HTML elements in your script. It basically says, let’s start programming in HTML now.
G’day Mate This is the text that you are displaying on the page.
</html> The / before a tag is indicative of the end of a tag. It basically says, let’s stop programming in HTML now.

Let’s look at another similar example.

Code <!DOCTYPE html>
<html>
<h1>G’day Mate</h1>
</html>
gday.html

G'day Mate




Solution

The “G’day Mate” is formatted differently.

The <h1> tag defines a heading. There are several different types of headings: <h1> to <h6>. There are also different tags for formatting text including <b>, <u>, and <i>.

Let’s take a step back. Open up Microsoft Word and recreate the following:

This is Heading 1.

This is Heading 2.

This is bold
This is underlined
This is bold, underlined and in italics


Utilising the styles pane at the top:


If you can understand how to format text in Microsoft Word, then formatting text in HTML will be a breeze. So let’s write a page that mimics the example above.

Code <!DOCTYPE html>
<html>
<h1> This is heading 1. </h1>
<h2> This is heading 2. </h2>
This is <b> bold </b><br>
This is <u> underlined </u>
<b><u><i><br> This is bold, underlined and in italics </b></u></i>
</html>
gday.html

This is Heading 1.

This is Heading 2.

This is bold
This is underlined
This is bold, underlined and in italics


Let’s explain what each line does:
Line Explanation
<h1> Start heading 1
</h1> Stop heading 1
<h2> Start heading 2
</h2> Stop heading 2
<b> Start bolding text
</b> Stop bolding text
<br> Line break, this adds a new line
<u> Start underline
</u> Stop underline
<i> Start italics
</i> Stop italics



Solution

The rest of the page would be formatted like h2.
