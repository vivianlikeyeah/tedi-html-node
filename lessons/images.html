Images

Meet Axon, the Axolotl. Axon will be our mascot for the remainder of this node. We’re going to create a webpage dedicated to Axon.

The details are as follows:
Title: The most underrated animal of all time.
Body: I present AXON the AXOLOTL


Code <!DOCTYPE html>
<html>
<head>
<title>The most underrated animal of all time.</title>
</head>
<body>
I present <b>AXON</b></u> the <b><u>THE AXOLOTL</b></u><br>
<img src="http://bit.ly/axonTEDI" height="200px" width="200px">
</body>
</html>
The most underrated animal of all time.
I present AXON the AXOLOTL


Let’s go through each new feature:
Element Explanation
<img> Defines an image. Does not have a closing tag.
src Src is an attribute that specifies the URL (web address) of the image. This can also be a local address like ‘image.png’
"http://bit.ly/axonTEDI" This is the URL of the image. If you enter it in your browser's address bar, the image will show up.
width This defines the width of your image. This value can be entered as a % of the width of the page or as a number of pixels.
height This defines the height of your image. This value can be entered as a % of the height of the page or as a number of pixels.
200px This sets the size to be 200 pixels.


Activity

Ask Paulo (or the friend of your choice) what their favourite animal is. Re-build the above utilising this animal. Save this file as animal.html
