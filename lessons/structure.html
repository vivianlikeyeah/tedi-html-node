Let's add some structure

Have you ever opened 10+ tabs at a time on your web browser? How did you navigate to the correct site if you couldn’t see it?

The trick is we can read a snippet of the websites title. This title is a part of what is called ‘metadata’. Metadata isn’t actually displayed in our HTML, however metadata serves a purpose to your site.

This includes the title of the webpage, information about it’s style, links to other scripts etc. Metadata is contained with a <head> tag. The rest of your website is contained within a <body> tag. Consider the following example:


Code <!DOCTYPE html>
<html>
<head>
<title> My Website </title>
</head>
<body>
<h1> This is heading 1. </h1>
<h2> This is heading 2. </h2>
This is <b> bold </b><br>
This is <u> underlined </u>
<b><u><i><br> This is bold, underlined and in italics </b></u></i>
</body>
</html>
My Website

This is Heading 1.

This is Heading 2.

This is bold
This is underlined
This is bold, underlined and in italics



Solution

We only added metadata, which isn’t displayed in our HTML.

Activity

Meet some new and ask ‘Hey can I make a website about you?’. Find a TEDI London Student and write a basic website about them. Allow them to rest assured that you will not be publishing this website so their data is safe. Save this page as hello.html.

Format the website as follows (assuming your new friend’s name is Paulo):

Paulo's Website

Paulo is a student at TEDI London.

They are 19 years old.
They are from Phoenix, Arizona.
Their favourite thing about London is the the Borough Markets.
