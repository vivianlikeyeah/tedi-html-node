Making connections

We’ve made a few static pages, but how do we start connecting them to build a website?
NOTE: If you’re using CodePen.io ensure you are in Project Mode.

Let’s make a search engine, we’ll call it Ping.


Code <!DOCTYPE html>
<html>
<head>
<title>Ping</title>
</head>
<body>
<h1> Ping </h1><br>
<a href="https://www.google.com" > Click here to search for something </a>
<body>
</html>

NOTE: This example may not work using codepen.io. Run it locally on your computer if this is the case.

Let’s go through our new feature:
Element Explanation
<a href=“https://www.google.com”> Creates a hyperlink. In this case the hyperlink leads to google.com. You may also use hyperlinks to link to local files that you have created.
</a> Ends the hyperlink.
